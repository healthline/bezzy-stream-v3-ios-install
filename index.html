Install on iOS
Open iOS app
iOS installation instructions
  1. Open this page in Safari on an iPhone or iPad.
  2. Tap Install on iOS and confirm the install prompt.
  3. After the app finishes installing, open Bezzy Depression from the Home Screen.
  4. If iOS says the enterprise developer is untrusted, open Settings.
  5. Go to General, then VPN & Device Management.
  6. Under Enterprise App, tap Healthline Networks, Inc.
  7. Tap Trust Healthline Networks, Inc., then return to Bezzy Depression.

The trust step is only needed the first time this enterprise developer is used on a device.

iOS does not allow an OTA-installed app to open automatically. Use Open iOS app after installation and trust are complete.

Android installation instructions
  1. Open this page in Chrome on an Android device.
  2. Tap Download Android APK.
  3. Open the downloaded file when Chrome finishes downloading it.
  4. If Android blocks the install, allow this browser to install unknown apps.
  5. Finish the install, then open Bezzy Depression from the app launcher.

Android may ask for install permission the first time a browser installs an APK.

This build is for authorized internal testing only.
