Install DEP Stage Open DEP Stage
  1. Open this page in Safari on an iPhone or iPad.
  2. Tap Install DEP Stage and confirm the install prompt.
  3. After the app finishes installing, open DEP Stage from the Home Screen.
  4. If iOS says the enterprise developer is untrusted, open Settings.
  5. Go to General, then VPN & Device Management.
  6. Under Enterprise App, tap Healthline Networks, Inc.
  7. Tap Trust Healthline Networks, Inc., then return to DEP Stage.
  8. This build is for authorized internal testing only.

The trust step is only needed the first time this enterprise developer is used on a device.

iOS does not allow an OTA-installed app to open automatically. Use Open DEP Stage after installation and trust are complete.
